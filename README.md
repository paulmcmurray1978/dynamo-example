# how to make a layer

make a new folder
cd into that folder
make a folder called nodejs
cd into it
run: npm init
hit enter until its finished
npm install what you need
go back a directory: cd ..
zip the nodejs folder
go to lambda layers, and create new layer

go to lambda, and add your new layer
